## Download

```sh
git clone git@github.com:AndreHammel/Desafio_01__Praticando_os_conceitos_do_ReactJS

```

```sh
cd Desafio_01__Praticando_os_conceitos_do_ReactJS

```

### Iniciar o projeto

```sh
npm install
```

ou

```sh
yarn
```

### Iniciar a aplicação

```sh
npm start
```

ou

```sh
yarn start
```
